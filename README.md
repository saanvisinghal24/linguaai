# LinguaAI 🌍
An AI-powered multilingual language learning platform with exam preparation for Goethe, DELF, JLPT and more.

## Features
- 🎙️ AI Speaking Partner (Whisper + Claude + ElevenLabs)
- ✍️ Writing Evaluator with official exam rubrics
- 📝 Grammar Checker with rule explanations
- 🧠 Flashcards with SM-2 spaced repetition
- 📊 Personalized AI study plans

## Tech Stack
Python · FastAPI · PostgreSQL · React · Anthropic Claude · OpenAI Whisper · ElevenLabs
